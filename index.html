Mini Windows 10

Windows dans votre poche!

Un petit Windows!

Mini Windows 10 pèse 1,5 GB.
Fonctionne sur clé USB ou DVD.

Mini Windows 10 ne démarre pas à partir de ton PC,
Il démarre a partir du fichier "boot.wim" sur ta clé USB/DVD.

Mini Windows 10 a tous les programmes pour remplacer
le Windows mort de ton PC!

Malencontreusement, Windows ne fonctionne plus!

Quelles applications sont installés?

Les applications sur Mini Windows 10 :

Explorateur de fichiers
Microsoft Paint
Bloc-notes
Visualisateur de photos
Outil capture d'écran
Enregistreur d'actions utilisateur
WordPad
Powershell
7-Zip
Speccy
VLC Media Player
Google Chrome
Editeur du registre
WinNTSetup4 (si vous voulez installer Windows 2000 jusqu'a Windows 11)
Invite de commandes

PS : Tu peux installer tes propres applications! mais elles seront perdus au redémarrage comme tes fichiers.

Pourquoi mes fichiers & applications disparraisent au redémarrage de Mini Windows 10?

Car Windows démarre sur la mémoire vive.

Au redémarage, la mémoire vive se vide.

Si tu veux garder tes fichiers / applications, enregistre-les sur ta clé USB/DVD Mini Windows 10 ou un disque dur externe.

Mémoire vive : 2GB minimum, 3GB recommendé.
Clé USB/DVD de 4GB minimum, 8GB recommendé.
Processeur : Verifier sur la configuration requise Windows 10.
